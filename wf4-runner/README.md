# WF4 Runner Service

Automated Docker deployment service for AI-Fabriek.

## Quick Start

```bash
# 1. Set GitHub token
export GITHUB_TOKEN=ghp_your_token_here

# 2. Start runner
docker-compose up -d wf4-runner

# 3. Check health
curl http://localhost:8787/health
```

## API Endpoints

- `GET /health` - Health check
- `GET /v1/status?buildId=xxx` - Build status
- `GET /v1/result?buildId=xxx` - Build result
- `POST /v1/build` - Trigger build
- `POST /v1/persistResult` - Persist result

## Environment Variables

- `PORT` - Server port (default: 8787)
- `GITHUB_TOKEN` - GitHub access token (required)
